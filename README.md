## Project Description
This project is based on the study of open-source software using Python as the selected software. The aim of this project is to understand the concept of open-source development, its philosophy, licensing, and its role in modern computing.

The project also includes practical implementation of Linux shell scripting to demonstrate basic system operations and automation tasks.

---

## About Python
Python is an open-source programming language that is widely used for software development, data science, web applications, and automation. It is known for its simple syntax and readability.

---

## Scripts Included

### Script 1 — System Identity Report
Displays system information such as user name, system details, kernel version, date, and uptime.

### Script 2 — Python Package Inspector
Checks whether Python is installed and displays its version.

### Script 3 — Directory Auditor
Displays information about important system directories including permissions and size.

### Script 4 — Log Analyzer
Takes user input and counts how many times a word appears in a sentence.

### Script 5 — Manifesto Generator
Generates a small open-source philosophy statement based on user input.
